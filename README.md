# My Portfolio 
## Getting started 
---
### Prerequisites
* npm
### Installing
Use `npm` package manager to 
* install the required dependencies:  
```
$ npm install  
```
* start the dev server:
```
$ npm start
```
* build for deployment:
```
$ npm run build
```
